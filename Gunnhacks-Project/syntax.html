If-Then

Format

{
 cond: then,
  …
 else
}

Example

4 -> x
{
  x < 5: out(7)
  out(9)
}
; prints out 7

While Loops

Format

[condition] (while true…)

Example

1 -> x
[x < 3] (
 1 + x > x
)
out(x) ; prints 3
